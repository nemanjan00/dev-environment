# Embedded Development Profile

## Toolchains

- **arm-none-eabi-gcc** / **arm-none-eabi-binutils** / **arm-none-eabi-newlib** — ARM bare-metal toolchain
- **platformio-core** — multi-platform embedded build system (supports Arduino, ESP-IDF, STM32, etc.)
- **avrdude** — AVR microcontroller programmer
- **esptool** — ESP8266/ESP32 flashing tool
- **cmake** — build system

## Debugging & Flashing

- **openocd** — JTAG/SWD debugging (supports STM32, ESP32, etc.)
- **stlink** — STM32 programmer/debugger
- **flashrom** — SPI flash chip read/write

## Hardware Interfaces

- **python-pyserial** — scriptable serial communication
- **picocom** — lightweight serial terminal
- **sigrok-cli** — logic analyzer and protocol decoder (I2C, SPI, UART, etc.)
- **i2c-tools** — I2C bus inspection and communication
- **dtc** — device tree compiler/decompiler
